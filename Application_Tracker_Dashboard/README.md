## üìä Description  

This **interactive Power BI dashboard** was designed to track and manage **job applications** efficiently. It helps job seekers and recruiters monitor the progress of applications across multiple companies, roles, and stages in the hiring pipeline.  

The dashboard highlights key insights such as:  
- **Applications by Status** ‚Üí Submitted, Interview, Offer, Rejected.  
- **Applications by Company/Role** ‚Üí Track where efforts are most concentrated.  
- **Timeline View** ‚Üí Monitor progress of applications over time.  
- **KPIs** ‚Üí Total applications, conversion rate (applications ‚Üí interviews), and offers.  

This project demonstrates how Power BI can be applied to **personal productivity and career management**, showing practical, real-world value.  

---

## ‚≠ê Features  

- **Interactive Filters** ‚Äì Drill down by company, role, date, and status.  
- **Conversion Metrics** ‚Äì Track interview-to-offer conversion rates.  
- **Application Timeline** ‚Äì Visualize the hiring pipeline stages over time.  
- **Responsive Layout** ‚Äì Optimized for both desktop and mobile.
